2、反作弊规则

1、关于游戏“客户端”/游戏“材质”

我们禁止使用违反游戏公平性的客户端与材质，包括但不限于（透视、反隐身、外挂、穿墙esp等）

我们允许仅修改游戏内容贴图、音效的客户端与材质

2、关于注入型工具

我们禁止一切注入型工具

3、关于言语发言与游戏内建筑

我们禁止一切网络暴力、黄、赌、毒词语，禁止一切脏话（包括但不限于：傻*）
我们禁止一切不文明游戏建筑（包括但不限于：“火箭模型”、不文明字样）

4、其他

我们禁止一切修改游戏数据、内存的物品、程序。

若违反以上反作弊规则将会被处以不等天数的封禁
